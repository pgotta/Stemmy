STEMMY v1.5 UPDATE SUMMARY
==========================

Stemmy v1.5 replaces the older cumulative overlay notes with one complete,
tested Windows package. The main changes include:

- hidden app-style launcher and maximized dedicated window;
- desktop shortcut and icon created by the installer;
- close-window shutdown and Settings > Close Stemmy;
- background GPU protection when minimized or unfocused;
- matched PyTorch, TorchVision and TorchAudio CUDA 12.8 stack;
- complete source/startup/CUDA/ONNX installer checks;
- chromatic tuner and local Chord Creator;
- improved Shazam and lyric lookup;
- persistent Karaoke sessions and retry-failed handling;
- safe individual dependency updates with rollback;
- Basic Pitch ONNX MIDI/Tab support without TensorFlow;
- Save As project export and live system monitoring.

For complete release notes, read RELEASE_NOTES.md.
For installation and repair instructions, read READ-ME-FIRST.txt and BUILD.md.
